1. Specifications & Usage

InsertDate (the "Software") is a resident utility. While running, pressing the Insert key inserts the current date as text characters. Multiple executable files (exes) are provided for different date formats, but the basic operation is identical; only the output format differs. You can manage the application by right-clicking the icon in the system tray to open the menu:

  • Start on user login: Controls automatic execution at login.
    • Clicking to enable this creates a shortcut in the user-specific startup folder: %AppData%\Microsoft\Windows\Start Menu\Programs\Startup and adds a checkmark to the menu.
    • Clicking again to disable it removes the shortcut from the folder and clears the checkmark.
  • Exit: Exits the Software and restores standard Insert key behavior.
  • Registry-free: The Software does NOT use the system registry for saving settings or startup registration.
  • No Network: The Software performs NO network communication and collects no personal information.
  • No Service: It does NOT register as a "Windows Service." It runs as a standard user process.
  • UAC: To type into Administrator-level apps, the Software must also be "Run as Administrator" (Windows Security Specification).
